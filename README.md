# bg3-SkillAndMod

博德之门 3 自制 Mod 与可复用开发 skill。

## 当前内容

- [`mod/MysticOrientalWeapons/`](mod/MysticOrientalWeapons/)：神秘东方武器集合。
- [`mod/SpiritKeeper/`](mod/SpiritKeeper/)：精灵球（Spirit Ball），捕获尸体并召唤为可控随从。
- [`skill/`](skill/)：已验证的 BG3 Mod 制作经验，以及可复用的 4154 条静态召唤预设与生成器。

## 目录约定

```text
mod/
├── ModName/
│   ├── ModName.pak
│   └── README.md
skill/
├── SKILL.md
├── references/
├── scripts/
└── assets/
```

仓库只保留可直接安装的 pak、简短说明和可复用 skill；解包目录、临时日志与机器专用缓存不提交。

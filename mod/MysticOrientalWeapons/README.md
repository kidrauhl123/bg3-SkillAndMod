# 神秘东方武器

这是一个包含两件原创武器的《博德之门 3》Mod。

## 内容

### 诸葛连弩

- 20 米远程重弩外观。
- 武器附魔随装备者等级成长：4/7/10 级获得 +1/+2/+3。
- 每回合首次命中额外造成等同于装备者等级的穿刺伤害。
- 附赠动作「机括连射」：每回合一次普通射击。
- 使用诸葛连弩击杀敌人后获得追击射击，可连续触发，不设每回合次数上限。

### 啥匕

- 随等级成长的传奇匕首。
- 每回合首次近战命中额外造成等同于装备者等级的穿刺伤害。
- 使用啥匕造成伤害时恢复 `1d6` 点实际生命值。
- 装备时任意击杀都能获得 2 回合隐身，不设每回合次数限制。

## 安装

将 `MysticOrientalWeapons.pak` 复制到《博德之门 3》的 `Mods` 目录，并在 Mod 管理器中启用。

BG3 Script Extender v30 或更新版本只用于自动把武器添加到主机背包；没有 Script Extender 时，武器本体数据仍在包内，但需要用其他方式获取物品。

# 精灵球

- 使用“收纳尸体”记录可处理的 NPC 尸体；收纳后尸体会消失。
- 使用“召唤精灵”打开列表，选择精灵后指定地面位置进行召唤。
- 召唤物作为可操控随从加入头像栏，保留模板外观、数值，并补回捕获时记录的技能与被动。
- 允许同时召唤多个不同精灵；驱逐由召唤物自带的原版技能完成。

当前版本预生成了 4154 个可召唤模板。特殊剧情脚本、对话、交易及捕获时的临时状态不保证保留。

## 剧情兼容性

本 mod 只允许收纳已经死亡的 NPC，不会控制或替换仍在参与剧情的角色。召唤出的精灵是独立的可操控随从，定位类似原版魔宠，仅用于移动与战斗；它不会继承原 NPC 的对话、交易或任务交互，也不会以原 NPC 的身份重新参与剧情。这样的设计将“控制 NPC 战斗”的玩法与原版剧情逻辑隔离，通常比直接把活着的 NPC 变成伙伴更加稳定。

收纳会让原尸体消失，因此建议先完成搜刮、死者交谈等尸体交互；极少数仍需检查尸体实体的特殊脚本不作保证。

本 mod 需要 Script Extender v30+。包文件名和内部 UUID 保持英文与稳定，以兼容旧存档和 Windows 工具链。

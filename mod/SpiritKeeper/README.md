# 东方御灵球（Spirit Keeper Orb）

当前原型只做三件事：

- 对尸体使用球体，记录原 NPC 的 GUID、名称和原版头像并将其收纳。
- 再次使用球体打开原版技能容器子菜单，按已收纳角色名称和头像召唤。
- 在子菜单中选择“收回当前随从”。

召唤时复用原 NPC 实体并调用 `AddPartyFollower`，因此原有装备、技能和被动会保留；随从会出现在原版队伍头像栏中并可被选中、移动和战斗。特殊 Boss 阶段脚本、对话和交易不保证，且同一角色只能被记录一次。

本 mod 使用 Script Extender v30+，记录保存在 mod variables 中。包文件名使用英文，避免 Windows/工具链对中文文件名的兼容问题。
